# Vault - API Key Manager

本地运行的 API Key 安全管理器，纯浏览器端加密存储，数据不离开本机。

## 快速开始

直接双击 `index.html` 即可运行，无需安装任何依赖。

> 需要支持 Web Crypto API 的现代浏览器（Chrome / Edge / Firefox / Safari）

## 项目结构

```
vault/
├── index.html   — 页面结构
├── style.css    — 样式表
├── app.js       — 业务逻辑
└── README.md    — 项目文档
```

## 功能概览

### 安全架构

- **主密码**是唯一核心认证，永远可用
- **手势密码**是可选快捷解锁方式，可随时开启/关闭/重置
- 加密算法：AES-256-GCM
- 密钥派生：PBKDF2（SHA-256，100,000 次迭代）
- 密码验证：PBKDF2 + SHA-256 哈希，不存储明文

### 首次使用

1. 创建主密码（需确认输入）
2. 可选设置手势密码（可跳过）
3. 进入主页面

### 登录方式

| 手势已开启 | 登录界面 |
|-----------|---------|
| 是 | 默认手势解锁，可切换到主密码 |
| 否 | 直接主密码输入 |

### API Key 管理

- 添加 / 编辑 / 删除
- 实时搜索（名称、URL、备注）
- 显示 / 隐藏 Key 值
- 一键复制到剪贴板

### 导出

- 导出所有 API Key 为 Excel（.xlsx）

### 设置（主页面右上角 ⚙）

| 功能 | 说明 |
|------|------|
| 修改主密码 | 验证旧密码后设置新密码，所有 API Key 自动重新加密 |
| 手势密码开关 | 随时开启/关闭手势快捷解锁 |
| 重置手势 | 验证主密码后重新绘制手势 |
| 重置所有数据 | 双重确认后清除全部数据 |

## 数据存储

使用 `localStorage` 存储：

| 键名 | 内容 |
|------|------|
| `vault_config` | 安全配置（密码哈希、盐值、手势配置） |
| `vault_data` | 加密后的 API Key 列表 |

## 安全说明

- API Key 使用 AES-256-GCM 加密存储
- 主密码通过 PBKDF2 派生 AES 密钥，不存储明文
- 手势密码仅存储 SHA-256 哈希，不可逆推
- 主密钥使用手势派生密钥包裹存储
- 修改主密码后手势自动关闭（安全考虑）
- 无后端、无网络请求、无数据上传
- 清除浏览器数据将导致所有内容丢失
